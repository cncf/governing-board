# RESOLUTIONS OF THE CNCF GOVERNING BOARD

Approved by email vote on April 9, 2026

## **Blanket License Exception for axe-core**

WHEREAS, the Headlamp project has requested an exception (Issue \#[1323](https://github.com/cncf/foundation/issues/1323)) to use [axe-core](https://github.com/dequelabs/axe-core), which is licensed under MPL-2.0.

WHEREAS, the CNCF Legal Committee recommends APPROVAL of a blanket exception to allow any CNCF project to use axe-core under MPL-2.0, subject to the following scope restrictions:

* **Separated Component:** axe-core must be used as a distinct module, not intermingled with the CNCF project’s own source files.

* **Unmodified**: The exception will apply only to the unmodified upstream version of the dependency.

RESOLVED, the Governing Board hereby APPROVES the recommended exception subject to the scope restrictions described in the foregoing recital.
